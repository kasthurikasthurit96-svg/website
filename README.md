**# Spice Garden Restaurant Website**

## Project Overview

Spice Garden is a modern and responsive restaurant website developed using HTML, CSS, and JavaScript.
The website provides an attractive user interface with food sections, menu display, image gallery, contact form, and smooth navigation.



# Features

* Responsive Navigation Bar
* Hero Section with Background Image
* Food Speciality Cards
* Restaurant Menu Section
* About Us Section
* Food Gallery
* Contact Form
* Smooth Scrolling Effect
* Mobile Responsive Design
* Interactive Hover Animations



# Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons
* Google Fonts

# How to Run the Project

1. Download or copy all files.
2. Save the files in one folder:

   * index.html
   * style.css
   * script.js
3. Open `index.html` in any web browser.
